### '''0-index.html'''

* Style: Poner 'style' dentro del header, footer, etc, nos permite definir color, altura, posicion, ancho, etc.

* Bottom: Bottom permite desplazar un elemento respecto a su posición original tomando como referencia el borde inferior del elemento. Esta
propiedad sólo afecta a los elementos cuya propiedad position tenga un valor de relative , absolute o fixed. En este caso usamos absolute, esto permite posicionar
elementos de manera absoluta, esto es de manera definida por valores de los atributos top, left, bottom y right, que indican la distancia con respecto a un punto.
